What is Oracle Cloud Infrastructure?

Getting Started with Oracle Cloud

Setting up your Oracle Cloud account is straightforward:

Glossary

vCPU (Virtual CPU): A virtual central processing unit that represents a portion of a physical CPU core allocated to a virtual machine.
Kubernetes: An open-source container orchestration platform that automates the deployment, scaling, and management of containerized applications.
Compute: The processing power and resources needed to run applications and workloads in the cloud.
Storage: The capacity to store and retrieve data, files, and applications in the cloud.
Networking: The infrastructure that enables communication between cloud resources and external networks.
Databases: Structured data storage systems that organize, store, and retrieve information efficiently.
Artificial Intelligence (AI): Technology that enables machines to perform tasks that typically require human intelligence.
